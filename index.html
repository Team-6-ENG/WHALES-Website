Overview

QUACK? is a campus-themed adventure game developed in Java using libGDX. Players must locate all pieces of a stolen duck statue without getting caught in order to prove their innocence.

Game Objective

  • Collect all duck statue pieces across the campus.
  • Finish before the timer reaches zero.
  • Final score is influenced by objectives and time remaining.

Controls

  • Move: WASD or Arrow Keys
  • Pause: P
  • Instructions: I
  • Music toggle: M
